Udemy Typescript: Class && Instance

Page: 홈
  • Object 객체: 코드로 작업을 수행하면서 사용할 수 있는 구체적인 요소

    객체의 장점: 클래스는 객체의 생성 속도를 높여주며, 객체 리터럴 표기법을 사용하는 것에 대한 대안이다.

  • 클래스 Class: 객체의 청사진

    장점: 동일한 구조와 메소드를 포함한 여러 객체를 쉽게 만들 수 있다.

  • 인스턴스 Instance: 클래스를 사용하여 객체의 형태, 포함해야하는 데이터, 클래스를 기반으로 객체를 쉽게 만들 수 있으려면 어떤 메소드가 필요한지 정의(Class내에 속하는 인스턴스라함)
